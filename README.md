# HR Analytics Attrition Dashboard

## Project Overview

This project analyzes employee attrition trends using Power BI. The dashboard helps HR teams understand workforce demographics, attrition patterns, job satisfaction, and educational backgrounds to support data-driven decision-making.

## Business Problem

Employee attrition impacts productivity, recruitment costs, and organizational performance.

The objective of this project is to:

- Analyze employee attrition trends
- Identify departments with higher attrition
- Understand workforce demographics
- Measure job satisfaction levels
- Support HR decision-making through interactive visualizations


## Tools Used

- Power BI
- Microsoft Excel / CSV
- Data Cleaning
- Data Visualization


## Dashboard KPIs

- Total Employees
- Attrition Count
- Attrition Rate
- Active Employees
- Average Age


## Dashboard Features

### Department-wise Attrition
Identifies departments with the highest employee turnover.

### Age Group Analysis
Analyzes employee distribution across age categories.

### Job Satisfaction Analysis
Compares satisfaction levels among different job roles.

### Education Field Analysis
Shows attrition trends across educational backgrounds.

### Gender Analysis
Visualizes attrition rates across age groups and gender.


## Key Insights

- Overall Attrition Rate: 16%
- Total Employees: 1470
- Active Employees: 1233
- Highest attrition observed in specific departments.
- Job satisfaction varies significantly across job roles.
